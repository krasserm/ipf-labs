Incubator and experimental stuff for the eHealth Integration Framework "IPF":http://github.com/krasserm/ipf. Currently contained are:

* A CouchDB-based message store and flow management service ("details":http://repo.openehealth.org/confluence/display/ipflab/Message+Storage+in+a+NoSQL+Database).  
* A Maven plugin for generating GDSL descriptors for IntelliJ IDEA.

h2. History

The project was originally developed at the "Gforge IPF Lab Subversion repository":http://gforge.openehealth.org/svn/ipf-lab/ (requires access with username _anonymous_ and empty password).  The project history has not been imported.
